# Chit Fund Management System

A production-grade financial management system for managing ₹5,00,000 chit funds with 20 members and 20-month duration.

## Features
- Layered backend architecture (Express)
- Month lifecycle management
- Payment tracking
- Lock enforcement
- Financial integrity rules
- Git version controlled

## Architecture
routes → controllers → services → repositories → database

## Tech Stack
- Node.js
- Express
- SQLite (temporary)
- GitHub version control
